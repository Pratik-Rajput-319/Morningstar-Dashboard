# 📊 Mutual Fund Performance Dashboard (Power BI)

## 📌 Overview

This project presents an interactive **Mutual Fund Analysis Dashboard** built using **Power BI**, designed to evaluate fund performance, risk, and portfolio composition.

The dashboard simulates real-world financial analysis similar to investment research platforms, combining key financial metrics, benchmarking, and portfolio insights in a single view.

---

## 🎯 Objective

* Analyze mutual fund performance using key financial metrics
* Compare fund returns against benchmark indices
* Understand asset allocation and sector diversification
* Build an interactive dashboard for decision-making

---

## 📊 Key Features

### 🔹 Fund Overview

* NAV (Net Asset Value)
* Expense Ratio
* Alpha & Beta
* Sharpe Ratio
* Standard Deviation
* AUM (Assets Under Management)

---

### 🔹 Risk & Rating Analysis

* Morningstar Rating (visual stars)
* Risk Level classification
* Medalist Rating

---

### 🔹 Asset Allocation

* Equity vs Debt distribution

---

### 🔹 Sector Allocation

* Financials
* IT
* Healthcare
* FMCG
* Others (which can be any)

---

### 🔹 Benchmark Comparison

* Fund vs Benchmark (Nifty 50 TRI)
* Returns comparison:

  * 1 Year
  * 3 Years
  * 5 Years

---

### 🔹 Interactivity

* Dynamic fund selection
* Cross-filtering across visuals
